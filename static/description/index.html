ALL POS Reports in Odoo

Using this module you can print many types of reports or receipts like, Sales Summary Report, Ongoing Session Report, Posted Session report from backend and Payment Summary, Category Summary, Product Summary, Order Summary, Session Audit Report, Inventory Audit Report from Point of Sales Screen in Odoo.

Features

Ongoing Session Report

Using Ongoing Session Report, you can generate report for opened or ongoing sessions.

Posted Session Report

Using Posted Session Report, you can generate report for Closed sessions.

Sales Summary Report

All/User wise Sales Summary Report.

Order Summary

Products Summary
